    KI-Zusammenfassung

    Empfohlen: python3 scripts/dev_server.py, Seite neu laden — die App erkennt den Server und aktiviert REST über dieselbe Origin automatisch (Well-known-URL). Kein CORS-OPTIONS an LM Studio. Alternativ: scripts/lm_studio_cors_proxy.py (siehe README). OpenAI- und Anthropic-Modus bleiben wählbar. HTTPS-Seite → API ebenfalls HTTPS („Mixed Content“).

    Tonfall und Schreibstil der Zusammenfassungen. Bei „Eigener Stil“ wird das Textfeld unten verwendet. Wirkt auf neu erzeugte Zusammenfassungen — bereits zwischengespeicherte ändern sich erst nach „Neu erstellen“.

    Wird verwendet, wenn oben „Eigener Stil“ ausgewählt ist. Beschreibe den gewünschten Tonfall/Schreibstil für die KI.

    Speicher in IndexedDB im Browser — bleibt nach Neustart des Webservers erhalten. 0 = unbegrenzte Gültigkeit (nur „Neu erstellen“ oder Modellwechsel erzwingt neu). Standard: 14 Tage.

    Es wird nur die fertige Zusammenfassung angezeigt — keine internen Reasoning-/Denkprozesse der KI.

    Beim automatischen Erzeugen und bei „Alle Zusammenfassungen“: wie viele Artikel parallel an die KI gehen. Höher = schneller, aber mehr Last für LM Studio / API. Bereich 1–16, Standard 4.

    Maximale Wartezeit pro KI-Aufruf (Zusammenfassung, „Alle Zusammenfassungen“, YouTube-Vorschläge usw.). Danach bricht der Browser die Anfrage ab. Bereich 15–3600, Standard 120.

    0 = aus. Sind alternative Links an, lädt die App echte Artikel über Bing News (RSS) über den lokalen Server — nicht die KI. App über scripts/dev_server.py öffnen. Anzeige siehe unten.

    Steuert, ob die Liste direkt unter der KI-Zusammenfassung sichtbar ist oder erst nach Klick auf „Weitere Quellen“ am Artikel.

    Steuert, ob Forumeinträge (z. B. Reddit) erst nach Klick geladen werden oder automatisch unter den Quellen erscheinen.

    Steuert, ob die KI-YouTube-Vorschläge erst nach Klick auf „YouTube“ erzeugt werden oder automatisch zusammen mit den KI-Zusammenfassungen vorab erstellt und gecacht werden.

    Pro Zeile eine Domain (z. B. example.com). Diese Domains werden in alternativen Links ausgeschlossen, inkl. Subdomains. URLs sind ebenfalls erlaubt.

    Öffnet eine Websuche zum Artikeltitel in einem neuen Tab (DuckDuckGo, Google oder Bing).
